Isolation Guarantees
Resource Mechanism Per-Agent Same-App Safe
Screen / Display CGVirtualDisplay (macOS SPI) Each agent gets an invisible monitor at a unique offset ✓ ✓
App Windows Cmd+N + window ID diff New window created per agent, tracked by CGWindowID ✓ ✓
Keyboard AXRaise + AppleScript keystroke Raise agent's window, then send to process — window receives ✓ ✓
Mouse / Clicks CGEvent at display coordinates Posted to virtual display offset — hits agent's window only ✓ ✓
Screenshots CGDisplayCreateImage or screencapture -l Captures only the agent's virtual display or specific window ✓ ✓
Clipboard NSPasteboard.pasteboardWithName_ Named pasteboard per agent — no cross-contamination ✓ ✓
User's Screen Virtual displays are invisible — user keeps full control No cursor movement, no focus steal, no visible windows ✓ ✓
Per-Agent Startup Sequence
01
Create Virtual Display
Spawn an invisible 1280×900 monitor via CGVirtualDisplay at a unique X offset (8000+).
02
Open App + New Window
Launch the app, send Cmd+N, diff window list to find the new CGWindowID.
03
Move Window to Display
Position window at the virtual display's bounds via AXUIElement position/size attributes.
04
Run Agent Loop
Claude computer_use + app_manager tools. Screenshots, clicks, keys all scoped to this agent's environment.
05
Cleanup
Destroy virtual display. macOS moves orphaned windows back to the main screen automatically.
